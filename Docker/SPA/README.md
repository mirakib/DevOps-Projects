# This folder contains docker based spa projects for polular frontend frameworks and technologies.

Each project demonstrates how to build and serve a static frontend application using different popular frameworks and technologies. The projects utilize Docker for containerization, ensuring that the applications can run consistently across various environments. Below is a brief overview of each project:
    
   [![Tech Stack](https://skillicons.dev/icons?i=react,vite,nodejs,npm,nginx,docker)](https://skillicons.dev)

1. **React Single Page Application (SPA) with Vite, Nginx & Docker**: This project showcases how to build and serve a React SPA using Vite for fast development and Nginx for production-grade static file serving, all containerized with a multi-stage Docker build.
   
[![Tech Stack](https://skillicons.dev/icons?i=angular,typescript,nodejs,nginx,docker)](https://skillicons.dev)
   
2. **Angular Static Frontend Application with Docker & Nginx**: This project demonstrates how to containerize an Angular application using Docker and serve it with Nginx, following best practices for building and deploying Angular apps in a containerized environment.
   
[![Tech Stack](https://skillicons.dev/icons?i=vue,nodejs,npm,yarn,nginx,docker)](https://skillicons.dev)

3. **Vue.js Static Frontend Application with Docker & Nginx**: This project illustrates how to build and serve a Vue.js application using Docker and Nginx, providing a streamlined approach to containerizing Vue.js apps for consistent deployment.
   
[![Tech Stack](https://skillicons.dev/icons?i=net,html,css,docker)](https://skillicons.dev)

4. **ASP.NET Core Static Frontend Application with Docker**: This project focuses on containerizing an ASP.NET Core application using a multi-stage Docker build, separating the build and runtime environments for optimal performance and security.
   
[![Tech Stack](https://skillicons.dev/icons?i=flask,python,docker)](https://skillicons.dev)

5. **Flask Static Frontend Application with Docker**: This project provides a hands-on introduction to Docker by containerizing a simple Flask web application, demonstrating the core concepts of Docker and ensuring consistent application behavior across different environments.
   
[![Tech Stack](https://skillicons.dev/icons?i=django,html,css,python,docker)](https://skillicons.dev)

6. **Django Static Frontend Application with Docker**: This project offers a practical guide to Docker with Django, showcasing how to package a Django web application into a Docker container for portability and consistent execution.
   
[![Tech Stack](https://skillicons.dev/icons?i=php,html,tailwind,docker)](https://skillicons.dev)

7. **PHP Static Frontend Application with Docker**: This project packages a minimal PHP static site inside a lightweight Alpine-based image using PHP's built-in development server, ideal for demos or lightweight prototypes.
   
Each project includes detailed implementation steps, Dockerfile configurations, and explanations of the technologies used, making it easy for developers to learn and apply Docker best practices for frontend applications.
